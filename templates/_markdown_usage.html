The page dropdown menu can be used to assign the article to another page

Use the position menu to change the position on the page, input the absolute number you want the article to appear at. For largely stylistic purposes some articles belong together, this is achieved by having a classname of "section" preceded by an id to uniquely identify a particular section group in the meta data field (see below) ie: "tvcoverage.section", moving these articles is possible but not advised.

The meta data field can be used to set an id and/or class on the article, not strictly a content management task, as it infers a knowledge of the DOM, but - in consultation with your friendly web administrator - it can be used to apply some light styling. Three options: a single word ie: "id": applies a unique id; a single word that starts with a period ie:".class": applies a single classname; two words separated by a period ie:"id.class": applies both.

The date field is required for new articles, the summary field is for internal use only and cannot be edited. The content field is where the real work is done, a rudimentary aquaintance with the Markdown language can only be underestimated, the links below will prove helpful.
